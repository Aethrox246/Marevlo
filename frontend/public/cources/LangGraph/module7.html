04 · Case Studies

Production Case Studies

Use Case

Klarna — AI Customer Service at Scale

Klarna’s AI assistant handled 2.3 million conversations in its first month, doing the work of 700 full-time agents.

It uses stateful context management, branches by intent, loops through clarification, and escalates to humans for complex issues.

Average resolution time dropped from 11 minutes to under 2 minutes.

Use Case

Elastic — Self-Refining Search Agent

Elastic builds search agents that iteratively refine queries, which is a direct Agentic RAG pattern.

The agent searches, evaluates if results are good enough, and if not, generates a better query and tries again until it is satisfied or reaches a limit.

Use Case

PGA Tour — Multi-Agent Content Generation

PGA Tour uses a multi-agent supervisor pattern for content generation:

  • Research agent gathers statistics
  • Writing agent creates articles
  • Validation agent fact-checks
  • Image agent selects photos

Result: 800 articles per week at $0.25 each, a 95% cost reduction.
